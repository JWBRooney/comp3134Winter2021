What is the definition of a server port?

A server port is a communication endpoint. 
It is used as the destination address for a packet of data.

What is server port range?

Server Port range is from 0 to 65353, 
with 0 to 1023 being reserved for common TCP/IP applications

How many bits does above represent?

A port number is 16 bits.
